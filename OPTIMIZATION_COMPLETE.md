## Usage Example: Modern API

```javascript
// app.easy
START SERVER 3000
USE MONGODB mongodb://localhost:27017/app

// Models
MODEL users {
  email: string
  name: string
  password: string
  role: enum[admin,user]
}

MODEL posts {
  title: string
  content: string
  author: reference(users)
  published: boolean
  views: number
}

// Authentication
AUTH users BY jwt
MFA users REQUIRED

// Routes (auto-generated REST + GraphQL)
GET /users FROM users
GET /users/:id FROM users
POST /users FROM users
PUT /users/:id FROM users
DELETE /users/:id FROM users

GET /posts FROM posts
POST /posts FROM users

// Validation
VALIDATE users {
  email: required:email
  password: required:min=8
  name: required:string
}

// Security
PROTECT /users
PROTECT /posts
RATE_LIMIT /users 100 per minute

// Real-time
SUBSCRIBE posts:created
SUBSCRIBE posts:updated

// GraphQL (auto)
GRAPHQL /graphql enabled

// Transactions
TRANSACTION users.create -> posts.create
```

**Result**: Complete REST API + GraphQL + WebSocket + Real-time with 20 lines of DSL
